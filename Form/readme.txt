Pasos para crear un servidor Node con Express:

1- Creamos un directorio para el proyecto y nos posicionamos dentro
2- Escribimos el siguiente comando: npn init (iniciamos el servidor)
3- Respondemos a las preguntas de especificaciones del proyecto (nombre,version,main...)
4- Instalamos Express con el siguiente comando: npm install express (esto añade la dependencia en el package.json y el node_modules)
5- Creamos un archivo .js con el nombre indicado en la propiedad "main" del package.json (archivo principal)
6- Para levantar el servido usamos el siguiente comando: node <nombre_main> (extension .js opcional)
7- Revisar los comentarios del código del archivo app.js para ver como usar Express.
